1. Export Backup

Export your entire database to a local JSON file. It is highly recommended to do this before running a Sync or Reset.

2. Database Sync

Pull down the latest components, firearms, loads, and targets from the master database. This will overwrite your existing records with the data from the master file.

3. Restore from Backup

Restore from a previous backup file. Merging will keep your existing data and only add/update records. Overwriting will wipe everything and replace it with the backup.

4. Session Download Export

Export a comprehensive zip file containing readable session data (JSON), database migration objects, and an Analysis plot image.

Hold Ctrl (Windows) or Cmd (Mac) to select multiple sessions.

Irreversible operations.
